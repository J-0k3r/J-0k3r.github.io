kali linux 安装

看完这篇，安装kali有手就行

准备

kali镜像下载：

https://www.kali.org/get-kali/#kali-installer-images

点3.6G这个按钮下载，如果下得慢，可以IDM多线程冲

下面这个是直接下链接

https://kali.download/base-images/kali-2023.1/kali-linux-2023.1-installer-amd64.iso

vm下载：

Download VMware Workstation Pro //官网 下载for Windows 版本
或者
https://download3.vmware.com/software/WKST-1700-WIN/VMware-workstation-full-17.0.0-20800274.exe //直接下载
vm 安装参考：
VMware Workstation PRO 17.0正式版+激活密钥 | 乐软博客

镜像部署

打开vm

选择自定义(高级)选项，点击下一步


选择【稍后安装操作系统】，然后点击下一步

客户机操作系统选择【Linux】，版本选择【Debian】，直接选择64位，我们选择Debian，选择完成之后继续点击下一步

命名虚拟机：关于虚拟机名称可根据自己喜好去改。位置选择C盘以外最好。完成之后点击下一步

处理器配置：根据自己的需求以及电脑的硬件限制合理进行配置。完成之后继续点击下一步

虚拟机的内存：这里同样根据自己的需求进行分配。然后点击下一步

网络类型：这里先选择第二项【使用网络地址转换(NAT)】，后期也可以更改，然后点击下一步



选择磁盘：这里选择创建新的虚拟磁盘，然后点击下一步

指定磁盘容量：同样这里根据自己的需求按需分配，分配好之后选择最下面一项【将虚拟磁盘拆分成多个文件】，然后下一步

默认点击下一步
然后
点击自定义硬件，iso为你前面下载的那个

然后点击完成

这时虚拟机配置完成了，接下来就可以直接安装Kali操作系统

先点击【开启此虚拟机】

点击【我已经完成安装】，将底下的黄色栏框消除
然后
选择Graphical install(图形化安装)

选择【Chinese(Simplified)-中文(简体)】，然后点击Continue

选择中国然后继续
选择语言汉语然后继续

【配置网络】这里输入一个主机名，然后继续

** 这里输入一个普通用户账号的用户名，然后继续**
输入一个账号的用户名，并且记住该用户名，然后继续
为刚刚的用户设置一个密码，然后继续

磁盘分区 这里选择【向导-使用整个磁盘】，然后继续

默认

这里分区方案选择推荐的第一个【将所有文件放在同一个分区中(推荐新手使用)】，然后继续
这里我们选择【结束分区设定并将修改写入磁盘】这个选项，然后点击继续

这里我们选【是】然后继续

下一步保持默认，直接点继续
然后等一会


安装完成点击继续
输入刚刚的用户名和密码点击登录就可以进入到桌面

设置root密码

进入系统之后，右键打开终端

输入以下命令

sudo -i #Linux终端命令下改变用户对命令使用权限的命令

然后输入登录密码，回车

然后

passwd root #这条命令表示修改root用户的密码

输入root用户密码（新密码）

然后重复输入root密码，接着回车，注意的是输入的密码不会显示。回车

如图所示即为成功

然后

sudo apt update

至此，kali安装完成

为了后续操作方便，建议继续做以下操作

更新apt以及换源：

终端输入

sudo vim /etc/apt/sources.list

输入密码进去后，按i 为输入模式

把原来的注释掉

然后粘贴下面这个上去

然后按esc，输入:wq! 回车

然后

sudo apt update && sudo apt -y full-upgrade

至此，更新完成
